# 日日是好日 — NichiNichi Café

**nichinichi-cafe.github.io** — the homepage of NichiNichi Café·Atelier in Olef, Eifel.

---

NichiNichi is a one-person micro-café in the village of Olef, Germany.  
No fixed menu. Everything fresh, everything made by hand.  
The name comes from 日々 — *every day* — and the Zen saying 日日是好日: *every day is a good day.*
